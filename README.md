# 🎰 Spin Pattern Tracker — Android App

## অ্যাপটি কী করে?
- আপনার স্ক্রিন স্বয়ংক্রিয়ভাবে স্ক্যান করে (OCR)
- BigWin / Super / Mega / Bonus ডিটেক্ট করে
- ফ্লোটিং উইন্ডোতে পরিসংখ্যান দেখায়
- গড় ব্যবধান ও ইতিহাস ট্র্যাক করে

## 📥 কিভাবে APK বানাবেন (বিনামূল্যে)

### পদ্ধতি ১ — Android Studio (সবচেয়ে ভালো)

1. **Android Studio ডাউনলোড করুন:**
   https://developer.android.com/studio

2. **এই ফোল্ডারটি খুলুন:**
   File → Open → SpinTracker ফোল্ডার সিলেক্ট করুন

3. **APK বানান:**
   Build → Build Bundle(s) / APK(s) → Build APK(s)

4. **APK পাবেন এখানে:**
   `app/build/outputs/apk/debug/app-debug.apk`

5. **ফোনে পাঠান:**
   WhatsApp বা Google Drive দিয়ে পাঠান, তারপর ইনস্টল করুন

---

### পদ্ধতি ২ — GitHub + Buildozer (অনলাইনে)

1. GitHub account বানান: https://github.com
2. এই কোড আপলোড করুন
3. GitHub Actions দিয়ে বিল্ড করুন
